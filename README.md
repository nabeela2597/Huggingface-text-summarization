**LLM Text Summarizer using Hugging Face**

This project demonstrates how to build a text summarization system using Large Language Models (LLMs) with the Hugging Face API. It uses powerful transformer models such as Pegasus and BART to generate concise summaries from long pieces of text. The project also shows how to customize summarization behavior by adjusting parameters like summary length, handle user interaction through Python scripts, and manage multiple models within a single codebase. It serves as a practical introduction to building NLP applications powered by LLMs.

**Key Features**

a) Text summarization using Pegasus and BART models
b) Integration with the Hugging Face API
c) Adjustable summary length (short or detailed summaries)
d) Python-based user interaction for input text
e) Support for multiple LLM models in one codebase
f)Response validation and structured output handling

**Technologies Used**

a) Python
b) Hugging Face Transformers API
c) Pegasus Model
d) BART Model
e) Natural Language Processing (NLP)
f) Large Language Models (LLMs)
